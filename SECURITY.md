# 安全说明

请不要把真实的数据库密码、讯飞星火密钥、Milvus token 或其他私密配置提交到仓库。

## 配置建议

- 使用 `.env.example` 作为模板，真实配置放在本地 `.env` 或部署平台的环境变量中。
- `SPARK_API_PASSWORD`、`MYSQL_PASSWORD`、`MILVUS_TOKEN` 等敏感字段应定期轮换。
- 生产环境应使用 HTTPS、强密码和最小权限数据库账号。
- 演示账号仅用于本地演示，正式部署前请修改或禁用。

## 漏洞反馈

如果发现安全问题，请通过仓库 Issues 说明影响范围和复现步骤。涉及密钥泄露、未授权访问或数据破坏的问题，请避免公开粘贴真实密钥、用户数据或服务器地址。
